Shopify App: Instant Bookstore - Scan And List ↗

Shopify Embedded Admin App | Javascript React Express Koa Node.js Apollo GraphQL

A work in progress

Description: Shopify web app to create product listings instantly for 11K+ indie bookstores in the U.S.

Market Need: only expensive (some $500+ per month) softwares exists for eCommerce storefronts.

Solves: allow anyone to deploy a Shopify bookstore rapidly with instant book listing & stock updating.

How It Works:

  • › Users can list and update stock simply by scanning the book's barcode, ISBN, or Amazon ASIN.
  • › Uses Shopify Admin GraphQL API to check against existing inventory and instantly updates stock.
  • › Calls an external Express Apollo server using GraphQL that acts as a API-Gateway & microservice.
  • › Server makes additional requests to external datasources for relevant book data.
  • › Historic pricing data, book description, title, author, category, is then relayed to client's React front-end.
  • › The user can review, edit book info, and easily add the book to their Shopify store within 2-3 seconds.
